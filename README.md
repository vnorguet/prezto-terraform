# prezto-terraform

Handy module for Terraform

## Features

* Completion
* Aliases

## Setup

1. Clone this repository
2. Add to `${ZDOTDIR:-$HOME}"/.zprezto/contrib`
3. Enable into `~/.zpreztorc`

    ```shell
    zstyle ':prezto:load' pmodule \
      ... \
      terraform
    ```

4. Reload your shell
